# 🏦 ICICI Bank Customer Segmentation Using K-Means Clustering

##  Project Overview

This project applies Machine Learning techniques to segment banking customers into meaningful groups based on their financial behavior and transaction patterns.

Using K-Means Clustering and Principal Component Analysis (PCA), customers were categorized into distinct segments to help improve customer targeting, personalization, marketing campaigns, and business decision-making.

---

##  Business Objective

Banks serve customers with different financial needs and behaviors. Treating all customers the same can reduce marketing effectiveness and customer engagement.

The objective of this project is to identify customer segments based on:

- Annual Income
- Monthly Spending
- Credit Card Usage
- Loan Amount
- Account Balance
- Digital Transactions
- Age

and generate actionable insights for customer-focused strategies.

##  Methodology

### Data Preparation
- Data cleaning and preprocessing
- Feature selection
- Standardization using StandardScaler

### Clustering
- K-Means Clustering Algorithm
- Elbow Method used to identify optimal clusters
- Selected 4 customer segments

### Dimensionality Reduction
- PCA applied to reduce dimensions
- Customer clusters visualized in 2D space

### Model Evaluation
- Silhouette Score used to evaluate clustering performance
